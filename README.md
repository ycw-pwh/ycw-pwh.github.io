# 个人网站项目

这是一个个人网站项目，使用现代前端技术栈构建。

## 项目结构

- `src/` - 源代码目录
- `node_modules/` - 依赖包目录
- `package.json` - 项目配置文件
- `tsconfig.json` - TypeScript配置文件

## 技术栈

- Node.js
- TypeScript
- PNPM 包管理器

## 开发

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev
```

## 部署

此项目部署在 GitHub Pages 上。
